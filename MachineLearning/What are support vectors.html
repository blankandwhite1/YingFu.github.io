def ExpSVOverC():
    X, y = make_blobs(n_samples=40, centers=2, random_state=6)
    
    X = np.vstack((X,[[7,-8]]))
    y = np.hstack((y,[0]))
    
    X = np.vstack((X,[[7,-6.3]]))
    y = np.hstack((y,[1]))
    
    for C in [0.01,0.1,1,10]:
        clf = svm.SVC(kernel='linear',C = C)
        clf = plotSVM(X,y,clf = clf,title = "C = " + str(C))

Output:

We can find that in the case of linearly separable data, the support vectors are those data points that

  1. lie exactly on the borders of the margins.
  2. fall inside the margin
  3. misclassifed points out of the margin

And we can also find that with C increased, the number of support vector decreased. The reason is mainly beacuse C is penality for unit violation, with the C is increased, a greater penalty is put on violation of the constraint, the solution will change to reduce the size of the violations, so the margin is made narrower, and less points fall inside it, so there are fewer support vectors.
