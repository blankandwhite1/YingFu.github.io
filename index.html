Optimization

The combination of Machine Learning and Optimization can do a lot of interesting things. Most of the problems in machine learning are essentially solving optimization problems. One typical example is SVM of which the optimization objective is a convex quadratic programming problem with linear constraints. In general, it is converted to solving dual problems which is easier to solve and can be naturally introduced into kernel function and then extended to the nonlinear classification problem. In addition, Machine learning can accelerate Optimization algorithms. So this part mainly covers some notes related to Optimization algorithms.
